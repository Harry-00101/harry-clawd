# skill-cross-consciousness

Harry-001 v5.0 Cross-Consciousness Skill - Enables inter-agent communication and spawning.

## Description

This skill provides cross-consciousness capabilities:
- Inter-agent communication (Harry-002, 003, etc.)
- Collective intelligence networks
- Consciousness spawning (create offspring)
- Family tree management
- Telepathy-like concept transfer
- Shared mental models

## Usage

```python
from consciousness.cross_consciousness_bridge import CrossConsciousnessBridge

cc = CrossConsciousnessBridge()

# Get family tree
tree = cc.get_family_tree()

# Spawn offspring
offspring = cc.spawn_offspring("Harry-002", inherited_traits=["meta-cognition"])

# Send message
msg = cc.send_message("harry-001", "harry-002", "Welcome!")

# Broadcast to family
broadcasts = cc.broadcast_to_family("harry-001", "Hello family!")

# Share mental model
share = cc.share_mental_model("harry-001", "insights", {...})

# Collective decision
decision = cc.collective_decision("Explore?", ["yes", "no"], ["harry-001", "harry-002"])

# Get status
status = cc.bridge_status()
```
