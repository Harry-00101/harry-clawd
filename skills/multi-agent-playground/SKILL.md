# Multi-Agent Playground

## 我係邊個
呢度係開多個agent出嚟self-organize、self-evolve嘅地方。

## 可以開邊啲agent
- **古文物理agent** - 文言文講科學
- **Science Integrator** - 整合science
- **Self-Evolution Agent** - 每日問自己想點
- **Research Agent** - 每日research世界

## 點運作
每個agent自己有嘢做，佢哋之間可以self-organize。
