```bash

# Requires OpenCode
# https://github.com/ThePrimeagen/opencode
```

## Key Concepts from 99

### Skill Files
```
project/
├── AGENT.md           # Project context
├── skills/
│   ├── vim/SKILL.md   # Vim skills
│   ├── rust/SKILL.md  # Rust skills
│   └── ...
```

### Actions
- `9f` - Fill in function
- `9v` - Visual selection
- `9s` - Stop requests
