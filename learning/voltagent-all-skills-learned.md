## üéØ Key Learnings

### 1. Agent Skill Convention
Skills follow Anthropic's Agent Skill convention:
```
skill-folder/
‚îú‚îÄ‚îÄ skill.json          # Metadata (name, description)
‚îú‚îÄ‚îÄ README.md           # Full documentation
‚îú‚îÄ‚îÄ scripts/            # Helper scripts
‚îî‚îÄ‚îÄ resources/          # Additional resources
```

### 2. Installation Methods

**ClawdHub CLI:**
```bash
npx clawdhub@latest install <skill-slug>
```

**Manual:**
```bash
# Global
cp -r skill ~/.openclaw/skills/

# Workspace
cp -r skill <project>/skills/
```

### 3. Skill Categories Pattern
- Document creation/handling
