## What is Fast-Agent?

MCP-enabled agent framework:
- **Complete MCP support** - Sampling, Elicitations, Streamable HTTP
- **CLI-first** - Terminal-based TUI
- **Declarative syntax** - Simple YAML configs
- **Multi-model** - Anthropic, OpenAI, Google, Azure, Ollama, Deepseek
- **Agent Skills** - Uses SKILL.md files!

## Key Features

| Feature | Description |
|---------|-------------|
| MCP Sampling | Server-initiated sampling |
| Elicitations | Human input requests |
| Streamable HTTP | Transport inspection |
| Structured Outputs | JSON/Pydantic |
| Vision/PDF | Multi-modal |
