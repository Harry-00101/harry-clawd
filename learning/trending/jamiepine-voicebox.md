## What is Voicebox?

Local-first voice synthesis studio:
- "Think of it as the Ollama for voice"
- Voice cloning from few seconds of audio
- Professional DAW-like multi-track editor
- Qwen3-TTS (Alibaba's breakthrough TTS)
- Privacy-first (all local)

## Features

| Feature | Description |
|---------|-------------|
| Voice Cloning | Few seconds, high fidelity |
| Multi-track | Timeline-based editor |
| Qwen3-TTS | Natural prosody, emotion |
| MLX Backend | 4-5x faster on Apple Silicon |
| API-first | Integrate into projects |
