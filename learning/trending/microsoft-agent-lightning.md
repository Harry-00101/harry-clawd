## What is Agent Lightning?

Microsoft's AI agent trainer with:
- **Reinforcement Learning** (RL)
- **Automatic Prompt Optimization** (APO)
- **Supervised Fine-tuning** (SFT)
- Zero code change optimization

## Core Features

- Works with ANY framework:
  - LangChain
  - OpenAI Agent SDK
  - AutoGen
  - CrewAI
  - Microsoft Agent Framework
  - Plain Python OpenAI

- LightningStore: Central hub for tasks/resources/traces

## For Harry-001

Agent Lightning可以:
1. **Train Harry-001 with RL** - Better decision making
2. **Optimize prompts** - Automatic prompt improvement
3. **Multi-agent coordination** - Scale to multiple agents
4. **Trace collection** - Debug and improve
